## GG CRM

	* Back-end based on perl Mojolicious framework - http://mojolicio.us
	* Front-end based on DHTML Suite Pane splitter - http://www.dhtmlgoodies.com

#### SYSTEM REQUIREMENTS

##### ADMIN BROWSER SUPPORT
	* Chrome 13+
	* Firefox 4+
	* Internet Explorer 10+
	* Safari 6+
	* Opera 12+

##### SERVER-SIDE
	* Perl 5.10.1+
	* Mysql 5+
	* utf8 encoding
